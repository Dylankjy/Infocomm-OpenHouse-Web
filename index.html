Lesson 2

How to store data in variables

A variable can store data, such as strings (sentences & words) as well as numbers.


For example, by running myVar = 2, the number 2 has been stored in the variable, myVar.

Variables can also be displayed simply by typing the variable name, myVar. It is also possible to add variables together for example:

a = 3 
b = 4 
a + b ### This is the same thing as saying 3 + 4

Lesson 3

If & else

This is quite self-explainatory. if this, do that, else, do the other. 

# Variables
favouriteFood = "vegetables"

# If statement
if favouriteFood = "vegetables":
    print("You must be very healthy!")
else:
    print("You should eat something healthier.")

Since favouriteFood = "vegetables", You must be very healthy! will be shown. Anything other than vegetables such as fried chicken, will print You should eat something healthier. instead.
